Every scorecard is a snapshot. Merged across 3 years, the same initiative can be tracked from the month it first appears to the month it's marked done — or to the month it quietly drops off. This page does that for all 1,601 line-items, collapsed into 260 distinct initiatives, so you can see what actually got finished and what never did — plus the 9-month hole in the town's own record.

① The Transparency Gap — why April–December 2025 are "unavailable"

The town page advertises a scorecard for every month — but the files for 9 straight months simply aren't there. The page lists a heading for each 2025 month (April, May, … December), but only January–March 2025 have an actual PDF behind the link. Every predictable URL for Apr–Dec 2025 returns 404 Not Found. The "monthly" transparency document goes dark right after March 2025 and doesn't resume until January 2026.

posted missing (404) existed but unlinked → recovered
What I found probing the town's file server directly:
• Apr–Dec 2025 (9 months): genuinely absent — no file at any naming pattern the town has ever used.
• June 2024: also missing (404).
• Oct 2023: never listed at all.
• Dec 2024: the page shows no working link, but the PDF does exist on the server (just un-linked) — I recovered it and included it here.
This is the period to CORA: the records either were never produced (a "monthly" report that stopped for most of a year) or exist but were never posted. The Dec 2024 case proves files can sit on the server unlinked — so Apr–Dec 2025 are worth a specific records request.
